# Database scripts

This folder holds database-related assets for the Ontario Tax Calculator:

- **migrations/** — Schema and data migrations (run in order)
- **seeds/** — Seed data for development or testing
- **schema/** — DDL or reference schema definitions

Add your SQL or other DB scripts here. Use a consistent naming scheme (e.g. `001_initial_schema.sql`, `002_add_tax_rates.sql`).
